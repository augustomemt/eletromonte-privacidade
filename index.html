Política de Privacidade — Eletromonte

Última atualização: 09/07/2025

1. Coleta de Dados

O Eletromonte não coleta, armazena nem compartilha dados pessoais dos usuários. Nenhuma informação como nome, e-mail, localização, fotos ou dados sensíveis é solicitada ou processada pelo aplicativo.

2. Autenticação/Login

O aplicativo exige login para garantir a segurança, mas não utiliza serviços de terceiros como Firebase ou Google Analytics. Os dados são utilizados apenas conforme previsto pela empresa responsável.

3. Compartilhamento de Dados

Não compartilhamos dados com terceiros, parceiros, anunciantes ou qualquer outra entidade.

4. Segurança

Adotamos práticas para garantir a segurança e integridade das funcionalidades do aplicativo.
